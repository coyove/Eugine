1. To submit URLs, just append them to your title (separated by space) and they will appear as: .
  2. Type "#post-num" in the text (e.g. #123) to refer a post. URLs automatically become clickable links.
  3. Use 3 backticks (```) at an exclusive line to wrap the code: 
    ```
    func main() {
        // code
    }
```
  4. Multiple spaces outside ``` block will be eliminated.
